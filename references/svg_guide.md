# 儿童绘本 SVG 插画指南

目标：画出来像绘本插画，而不是“几何图形示意图”。关键在于形状语言、配色、表情、层次和细节点缀。

## 画布与层次

统一使用：

```xml
<svg viewBox="0 0 800 600" xmlns="http://www.w3.org/2000/svg">
```

每页从后往前画 4 层：

1. **天空/背景：** 大色块或柔和渐变，可加太阳、月亮、云。
2. **远景：** 山丘、树林、房子；低饱和、形状简单。
3. **地面/中景：** 主角站立或活动的场景。
4. **主角与近景：** 最大、最鲜艳，位于画面中心或横向三分之一处。

## 形状语言：以圆润为主

儿童绘本不宜有尖锐轮廓：

- 身体优先用椭圆、圆形和圆角矩形。
- 必须有尖形（刺猬刺、树）时，整体仍要敦实、圆润。
- 路径优先使用 Q/C 曲线，而不是直线折线。
- 云朵可由 3–4 个重叠圆组成。
- 树可用波浪形树冠和粗短树干。

## 脸是角色的核心

- 眼睛：黑色小圆 + 小白高光圆。
- 腮红：两个半透明粉色圆。
- 嘴：用 Q 曲线表现开心或难过；惊讶可用小圆。
- 表情必须和文案一致。文字是紧张、难过或惊喜，脸也要表现出来。

## 配色

全书保持一个 6–9 色的低饱和暖色调色板。示例：

- 底色：`#fdf6ec` 奶油、`#dceefb` 淡天蓝、`#2c3e6b` 夜蓝。
- 主角色：`#e8985a` 暖棕橙、`#8fbf6b` 草绿、`#f2b64f` 杏黄。
- 点缀：`#f4766e` 珊瑚红、`#ffd9a0` 浅杏。

角色颜色必须锁定。第 1 页刺猬身体是 `#a8764f`，第 8 页也必须是同一个颜色。

## 保持角色一致性

第一次画主角时，先保留一份角色 SVG 代码片段。在之后每页复制它，只改：

- `transform`：位置、缩放或翻转。
- 表情。
- 四肢姿势。

最终文件中，不要在多页 SVG 重复使用同一个 `id`，例如 `id="hero"`。角色容器使用 `class="hero"` 即可；`id` 只用于需要引用的渐变、裁剪或滤镜，并且**每一页都必须唯一**。

例如：

```xml
<g class="hero" transform="translate(320,420)">
  <!-- 主角形状 -->
</g>
```

如果使用渐变，ID 要按页面编号区分：`sky-cover`、`sky-01`、`sky-02`。因为所有 SVG 会出现在同一个 HTML DOM 中，重复 ID 会导致渐变或滤镜串页。

## 细节点缀

每页加 2–3 个细节就够：小草、飘落的叶子、花瓣、星星、蝴蝶，或主角脚下透明的扁椭圆阴影。没有阴影时，角色很容易像漂浮在空中。

## 避免

- 纯几何堆砌：三角形 + 方块 = 房子，过于潦草；至少加门、窗和屋顶细节。
- 在 SVG 里写大段文字。文字属于页面文案区；插画中最多保留简短拟声词气泡。
- 过细的描边风格。以色块填充为主；描边少用或用同色系深一档。
- 荧光色或纯黑大面积。
- `<script>`、`<foreignObject>`、外部图片或网页链接。它们会被构建脚本拒绝。

## 单页骨架示例

```xml
<svg viewBox="0 0 800 600" xmlns="http://www.w3.org/2000/svg">
  <defs>
    <linearGradient id="sky-01" x1="0" y1="0" x2="0" y2="1">
      <stop offset="0" stop-color="#dceefb"/>
      <stop offset="1" stop-color="#fdf6ec"/>
    </linearGradient>
  </defs>
  <rect width="800" height="600" fill="url(#sky-01)"/>
  <circle cx="680" cy="90" r="46" fill="#f2b64f"/>
  <ellipse cx="400" cy="640" rx="520" ry="160" fill="#b5d69a"/>
  <ellipse cx="320" cy="520" rx="90" ry="14" fill="#000" opacity="0.1"/>
  <g class="hero" transform="translate(320,420)">
    <!-- 主角，逐页复用这组形状 -->
  </g>
</svg>
```
